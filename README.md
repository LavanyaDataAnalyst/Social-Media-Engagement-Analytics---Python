## 📘Project Overview:

* Analyze social media dataset with 5000+ records
* Study engagement trends across posts and users
* Perform sentiment-based and behavior-based analysis
* Generate insights to improve content strategy and performance
  
## 🗂️ Dataset Description

* Dataset contains social media engagement data
* Includes user details, post details, and engagement metrics
* Captures likes, comments, shares, impressions, and watch time
* Also includes sentiment, hashtags, and device information
  
## 🧱 Data Structure

* user_id → Unique user identifier
* post_id → Unique post identifier
* age, gender, country → User demographic details
* post_type, post_category → Content classification
* likes, comments, shares → Engagement metrics
* impression_count → Post visibility count
* sentiment → Positive / Negative / Neutral
* device_type → Device used for access
* engagement_rate → Overall engagement percentage

## 🛠 Tools & Technologies:
>* **Python** → Core programming language
>* **Pandas** → Data manipulation and analysis
>* **NumPy** → Numerical computations
>* **Matplotlib** → Basic data visualization
>* **Seaborn** → Statistical data visualization
>* **Plotly** → Interactive visualizations

## ⚙️ Features Implemented:

* Data loading and preprocessing using Pandas
* Feature engineering (engagement score, hashtag count)
* Categorization of likes, comments, and shares
* Sentiment-based engagement analysis
* Country-wise and age-wise trend analysis
* Verified vs non-verified account comparison
* Time-based engagement pattern analysis
* Visualization using bar charts, heatmaps, and scatter plots

## 📈 Key Insights
* Video/reels content performs best in engagement
* Younger users show higher activity levels
* Positive sentiment posts generate higher engagement
* Mobile users have higher watch time
* Certain countries show significantly higher engagement rates
* Peak engagement occurs during morning and evening hours

## 💡 Recommendation
* Focus on video and reel-based content for better reach
* Target high-engagement countries for marketing campaigns
* Post content during peak activity hours
* Optimize content for mobile users
* Encourage positive sentiment content creation
* Use verified accounts to improve credibility and reach
